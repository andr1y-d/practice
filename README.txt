Структура файлів

index.html — основний HTML-файл, що містить сучасну та інтуїтивно зрозумілу розмітку форми.

styles.css — файл з адаптивним дизайном для екранів шириною 320 та 1280 px та шрифтом

main.js — файл зі скриптами для плавної та зручної взаємодії з формою.


Як запустити?

Скачайте або клонуте репозиторій.

Переконайтеся, що всі файли знаходяться у відповідних директоріях.

Відкрийте index.html

Використаний Tailwind CSS для стилізації
Використані Flexbox
